# 수종 도감 (Species Catalog)

식물 수종별 **개화시기 · 단가표 · 수급처**를 데이터로 축적하고, 필터·검색·정렬로 조회하는 정적 웹앱입니다.
브라우저에서 직접 **추가·수정·삭제** 가능하며, 저장은 localStorage에, 백업/공유는 **JSON 파일 내보내기/가져오기**로 처리합니다.

어서화(꽃 중개 플랫폼)와 완전히 분리된 독립 모듈로, 서버·빌드 도구 없이 정적 서버만 있으면 동작합니다.

## 폴더 구조

```
species-catalog/
├── index.html          # UI: masthead + 필터 rail + 카드 그리드 + 편집 모달
├── css/style.css       # 스타일 (라이트/다크 자동)
├── js/app.js           # 상태·필터·CRUD·저장·임포트/익스포트
├── data/species.json   # 최초 시드 (12개 예시)
└── README.md
```

## 실행

`fetch()`로 시드 JSON을 읽으므로 로컬 서버로 열어야 합니다.

```bash
cd species-catalog
python3 -m http.server 8080
# 브라우저에서 http://localhost:8080
```

또는 VS Code Live Server, `npx serve` 등 정적 서버 아무 것이나 무방합니다.

## 데이터 관리

### 최초 로드
1. `localStorage["species-catalog:v1"]`가 있으면 → 그 값을 씀
2. 없으면 → `data/species.json`을 시드로 저장

### 추가·수정·삭제
- 우상단 **+ 수종 추가** 버튼으로 모달을 열어 신규 등록
- 카드에 마우스를 올리면 우상단에 **✎(수정)**, **✕(삭제)** 아이콘이 나타남
- 저장 즉시 localStorage에 반영

### 백업 / 공유
- **내보내기**: 현재 데이터를 `species-catalog-YYYY-MM-DD.json` 파일로 다운로드
- **가져오기**: 저장한 JSON 파일을 업로드 → 현재 데이터를 대체
- **시드 복원**: 모든 사용자 데이터를 지우고 `data/species.json`으로 리셋

### 마스터 목록 (카테고리·색상)
- 모달에서 카테고리는 드롭다운으로 선택하거나 **새 카테고리** 입력창에 타이핑
- 색상은 기존 칩 클릭 또는 하단 입력창에 새 값 추가
- 새로 넣은 값은 자동으로 마스터에 편입되어 이후 필터·모달에서 재사용

## 필터

| 필터 | 동작 |
|---|---|
| 검색 | 수종명/학명 부분일치 (대소문자 무시) |
| 개화월 | 다중 선택. 하나라도 겹치면 매칭 |
| 카테고리 | 다중 선택 (OR) |
| 개화색 | 다중 선택. 하나라도 겹치면 매칭 |
| 수급처 | 단일 선택. 데이터에서 자동 수집 |
| 가격 범위 | 단가표 항목 중 하나라도 범위 내면 매칭 |

## 데이터 스키마

```jsonc
{
  "categories": ["교목", "관목", ...],
  "colors": ["백색", "황색", ...],
  "species": [
    {
      "id": "sp-001",                        // 자동 부여 (편집 시 유지)
      "name": "왕벚나무",                    // 필수
      "scientificName": "Prunus × yedoensis",
      "category": "교목",                    // categories 중 하나
      "bloomMonths": [3, 4],                 // 1~12
      "colors": ["백색", "분홍"],
      "prices": [
        { "spec": "R6", "unit": "주", "price": 45000 }
      ],
      "suppliers": [
        { "name": "천리포수목원", "region": "충남 태안", "contact": "041-672-9982" }
      ],
      "notes": "가로수·공원용으로 다용."
    }
  ]
}
```

## 참고

`data/species.json`의 시드 데이터(수종·가격·수급처)는 **구조 예시용 목업**입니다. 실제 시세와 실 취급 여부는 검증되지 않았으므로 실무 발주에는 사용하지 마세요. 정확한 데이터는 국립수목원, 조달청 나라장터, 지역 산림조합 등의 자료로 대체하시기 바랍니다.
