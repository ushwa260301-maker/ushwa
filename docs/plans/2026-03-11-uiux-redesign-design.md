# 어서화 UI/UX 리디자인 - 배민 스타일

> **Goal:** 배달의민족, 요기요, 여기어때, 무신사를 참고한 전면 UI/UX 리디자인. 투박한 현재 디자인을 세련되고 감성적인 배민 스타일로 전환.

## 디자인 방향

**핵심 키워드:** 감성적, 부드러운, 친근한, 대담한 타이포그래피

### 컬러 시스템 (유지 + 보강)

기존 Primary #E91E63 유지하되 활용도를 높임:
- Primary: `#E91E63` (메인 CTA, 강조)
- Primary Soft: `#FFF0F5` (배경 틴트, 카드 호버)
- Primary Gradient: `#E91E63` → `#F06292` (히어로, 배너)
- Surface: `#FFFFFF` (카드)
- Background: `#FAFAFA` (페이지 배경 - 좀 더 뉴트럴하게)
- Text Primary: `#111111` (제목)
- Text Secondary: `#666666` (본문)
- Text Muted: `#999999` (보조 정보)
- Success: `#00C853` (주문 완료)
- Warning: `#FF9800` (주의)

### 타이포그래피

Pretendard 유지, 사이즈/웨이트 체계 개선:
- 히어로 제목: 28px / Bold (800)
- 섹션 제목: 20px / Bold (700)
- 카드 제목: 15px / SemiBold (600)
- 본문: 14px / Regular (400)
- 캡션: 12px / Regular (400)
- 뱃지: 11px / Medium (500)

### 간격 체계

- 섹션 간: 32px
- 카드 간: 12px
- 카드 내부 패딩: 16px
- 화면 좌우 패딩: 20px (모바일), 24px (태블릿+)

## 컴포넌트별 리디자인

### 1. 헤더

**현재:** 로고 + 검색바 + 아이콘들 (일자 나열, 밋밋)

**변경:**
- 1단: 로고 + 위치(📍서울 강남) + 알림/장바구니
- 2단: 풀폭 검색바 (둥근 pill shape, bg-gray-100)
- 배경: 화이트, 하단에 미세한 그림자 (shadow-sm)
- 높이: 고정 sticky, 스크롤 시 검색바 숨김 (선택)

### 2. 바텀 네비게이션

**현재:** 텍스트 + 아이콘 기본 배치

**변경:**
- 아이콘: Lucide에서 더 둥근 스타일 선택 (Home, Search, ShoppingBag, User)
- 활성 상태: Primary 색상 + 상단에 3px 인디케이터 바
- 높이: 60px → 좀 더 터치 편안하게
- 배경: 화이트 + 상단 그림자
- 라벨: 10px, 활성시만 Primary색

### 3. 홈페이지 히어로 배너

**현재:** 단색 그라데이션 + 텍스트 (단조로움)

**변경:**
- 풀폭 라운드 카드 (16px radius)
- 그라데이션: 좌→우 핑크 (#E91E63 → #F48FB1)
- 대담한 타이포: 28px Bold "오늘의 꽃 한 송이"
- 우측에 꽃 이모지/일러스트 장식
- 하단에 인디케이터 도트 (슬라이더 형태)

### 4. 카테고리 그리드

**현재:** 4x2 그리드, 원형 아이콘

**변경:**
- 수평 스크롤 (1줄)
- 각 항목: 56px 원형 아이콘 + 라벨
- 아이콘 배경: 파스텔 그라데이션 (각 카테고리별 다른 색)
- 간격: gap-16px
- 선택 시: 약간 scale-up + 하단 Primary 인디케이터

### 5. 꽃집 카드 (ShopCard)

**현재:** 작은 이미지 + 텍스트 나열 (투박)

**변경:**
- 이미지 영역: aspect-[4/3], 라운드 12px, 오버레이 그라데이션 (하단 어둡게)
- 이미지 위 뱃지: "영업중" (초록), "인기" (Primary pill)
- 이름: 15px SemiBold
- 평점: ⭐ 4.9 (256) - 한 줄
- 배달 정보: "배달 30~50분 · 배달비 3,000원" 한 줄 캡션
- 전체 카드: bg-white, rounded-xl, border 없음, shadow-sm
- 호버: shadow-md + translateY(-2px) 부드러운 리프트

### 6. 상품 카드 (ProductCard)

**현재:** 정사각 이미지 + 간단한 텍스트

**변경:**
- 이미지: aspect-square, rounded-xl
- 할인 뱃지: 좌상단 빨간색 라운드 뱃지 "16%"
- 꽃집명: 12px 회색
- 상품명: 14px SemiBold, 2줄 clamp
- 가격: 16px Bold Primary색 (할인가) + 취소선 원가
- 리뷰: 작은 ⭐ + 숫자
- 전체: bg-white, rounded-xl, 그림자 없음 (깔끔)

### 7. 섹션 헤더

**현재:** h3 텍스트 + "더보기" 링크 (밋밋)

**변경:**
- 제목: 20px Bold + 이모지 (🔥 인기 꽃집, ✨ 추천 상품)
- 서브타이틀: 13px 회색 ("지금 가장 사랑받는 꽃집이에요")
- "더보기 >" 우측 정렬, Primary 색상
- 간격: mb-16px

### 8. 검색 페이지 (꽃집 목록)

**변경:**
- 상단: 카테고리 필터 (수평 스크롤 pill chips)
- 정렬: 드롭다운 또는 탭 형태 (평점순/거리순/이름순)
- 카드: 2열 그리드, gap-12px
- 무한 스크롤 또는 "더보기" 버튼

### 9. 꽃집 상세 페이지

**변경:**
- 헤더 이미지: 풀폭, aspect-[2/1], 하단 그라데이션 오버레이
- 꽃집 정보: 이미지 아래 오버랩 카드 (음수 마진으로 겹침)
- 평점: 크게 표시 (⭐ 4.9 · 리뷰 256개)
- 탭: 밑줄 스타일 (shadcn tabs line variant)
- 메뉴 탭: 2열 상품 그리드

## 애니메이션 & 인터랙션

- 페이지 진입: 부드러운 fade-in (200ms)
- 카드 호버: translateY(-2px) + shadow 전환 (150ms)
- 탭 전환: 밑줄 슬라이딩 애니메이션
- 스켈레톤: pulse 애니메이션 유지
- 바텀 네비: 활성 아이콘 약간 scale-up (1.1)

## 구현 범위

1. **globals.css** - 배경색, 타이포 체계 조정
2. **header.tsx** - 2단 구조 + 위치 표시
3. **bottom-nav.tsx** - 인디케이터 바 + 크기 조정
4. **page.tsx (홈)** - 히어로/카테고리/섹션 전면 리디자인
5. **shop-card.tsx** - 카드 디자인 대폭 개선
6. **product-card.tsx** - 카드 디자인 개선
7. **shops/page.tsx** - 목록 페이지 레이아웃 개선
8. **shops/[id]/page.tsx** - 상세 페이지 리디자인
