견적서 · 스케줄 · 실비명세서 — 로우데이터(재고·단가) 자동 연동

로우데이터 (자재/재고)

여기에 등록한 품목과 단가가 견적서·실비명세서에서 선택·자동계산됩니다.

잔여재고 = 재고수량 − 실비명세서 실사용수량 합계. 마이너스면 빨간색으로 표시됩니다.

견적서

품목을 선택하면 단가가 자동 입력됩니다. (단가는 직접 수정 가능)

스케줄

공정/작업 일정과 진행 상태를 관리합니다.

실비명세서

실제 사용 수량을 입력하면 실비금액과 견적 대비 차액이 자동 계산됩니다.

차액 = 견적금액 − 실비금액. 양수(절감)는 초록, 음수(초과)는 빨강.

데이터는 이 브라우저에 자동 저장됩니다. 다른 기기/사람과 공유하려면 「전체 백업(JSON)」으로 파일을 내보내 전달하세요.
